🔑 Clave de API de Anthropic

Para que el análisis de foto y las recetas de respaldo por IA funcionen fuera de claude.ai, necesitas pegar aquí tu propia clave de la API de Anthropic. Se guarda solo en este navegador (nunca sale de tu dispositivo salvo hacia la API de Anthropic) y solo tú la usas.

¿No tienes clave? Créala en console.anthropic.com → API Keys. Sin ella, la app sigue funcionando en modo manual (sin foto por IA).
